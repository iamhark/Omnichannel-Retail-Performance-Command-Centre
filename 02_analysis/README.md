# Analysis

Use this area for Excel/Python work, reproducible transformations, and generated exhibits.

- `code/` - reusable Python scripts or documented Excel logic.
- `notebooks/` - exploratory and explanatory notebooks.
- `data/processed/` - derived/cleaned data only.
- `outputs/figures/` and `outputs/tables/` - generated material used in synthesis and deliverables.

Do not place final reports or slide decks here.
